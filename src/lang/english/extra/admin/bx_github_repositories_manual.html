Manual for Shop Operators

This module connects your shop to a GitHub App. Then you load repositories, activate the required entries, and sync ZIP files into the download folder. You can automatically create products for your shop. To do this, create a template product in the desired category and enter its product ID in the module parameters. All the settings you make in this product will be considered when creating the products.

Quick Start

  1. Open the setup assistant and enter App ID, Installation ID, and PEM file.
  2. Save settings and test the connection.
  3. Load/update repositories.
  4. Activate the required repositories and save the selection.
  5. Download releases or update ZIP per repository.
  6. Create products or delete existing ones.

Important Note on Tags

Each repository requires at least one version tag (e.g., v2.0.5). Without a tag, no ZIP file can be loaded.

git tag -a v2.0.5 -m "BX Products Video v2.0.5"
git push origin v2.0.5

ZIP Status in the Tab

  • Green dot: ZIP is present in the download folder.
  • Red download button: A download is required for the active repository.

Download Link in the Table

The local file name is clickable for the shop owner (Admin) and uses the Admin endpoint for secure download from the download folder.
