Manual for shop operators

This module connects your shop to a GitHub App. Then you load repositories, activate the required entries, and sync ZIP files into the download folder.

Quick start

  1. Open the setup assistant and enter App ID, Installation ID, and PEM file.
  2. Save settings and test the connection.
  3. Load/update repositories.
  4. Activate required repositories and save the selection.
  5. Download releases or update ZIP per repository.

Important note about tags

Each repository needs at least one version tag (for example v2.0.5). Without a tag, no ZIP file can be downloaded.

git tag -a v2.0.5 -m "BX Products Video v2.0.5"
git push origin v2.0.5

ZIP status in the tab

  • Green dot: ZIP exists in the download folder.
  • Red dot: ZIP is missing, download is recommended.
  • Red download button: download is required for an active repository.

Download link in table

The local file name is clickable and uses the admin endpoint for secure downloads from the download folder.
