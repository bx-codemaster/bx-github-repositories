Handbuch für Shopbetreiber

Dieses Modul verbindet Ihren Shop mit einer GitHub App. Danach laden Sie Repositories, aktivieren die gewünschten Einträge und synchronisieren ZIP-Dateien in den Ordner download. Sie können automatisch Produkte für Ihren Shop erstellen. Dazu legen Sie ein Template-Produkt in der gewünschten Kategorie an, dessen Produkt-Id hinterlegen Sie bei den Modulparametern. All die Vorgaben, die Sie in diesem Produkt tätigen, werden beim Erstellen der Produkte berücksichtigt.

Schnellstart

  1. Setup-Assistent öffnen und App ID, Installation ID sowie PEM-Datei eintragen.
  2. Einstellungen speichern und Verbindung testen.
  3. Repositories laden/aktualisieren.
  4. Benötigte Repositories aktivieren und Auswahl speichern.
  5. Releases herunterladen oder pro Repository ZIP aktualisieren.
  6. Produkte erstellen oder bestehende wieder löschen.

Wichtiger Hinweis zu Tags

Jedes Repository benötigt mindestens einen Versions-Tag (z. B. v2.0.5). Ohne Tag kann keine ZIP-Datei geladen werden.

git tag -a v2.0.5 -m "BX Products Video v2.0.5"
git push origin v2.0.5

ZIP-Status im Tab

  • Grüner Punkt: ZIP ist im Download-Ordner vorhanden.
  • Roter Download-Button: Für aktives Repository ist ein Download erforderlich.

Download-Link in der Tabelle

Der lokale Dateiname ist für den Shopinhaber (Admin)anklickbar und nutzt den Admin-Endpoint für den sicheren Download aus dem Download-Ordner.
